## 使用插件

[weapp-cookie](https://github.com/charleslo1/weapp-cookie)
> 一行代码让微信、头条、百度、支付宝小程序支持 cookie

[mybatis-generator-plugin](https://github.com/itfsw/mybatis-generator-plugin)
> MyBatis Generator 插件的拓展插件包

- 广场
	- 晒娃
	- 嗮物

- 宝宝

	- 饮食记录 ✅
	- 体重记录 ✅
	- 身高记录
	- 预防记录
